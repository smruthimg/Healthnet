Number Filters

Select the number of each type of filter you have selected. For example: Choosing 1 will allow you to enter one of the type of filter. If the type is Username, you will be able to filter using one username. If you select more, you will be able to filter using that amount of usernames. For example: 5 usernames will show the logs for all 5 usernames.
